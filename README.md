Machine Learning Algorithmen und Übungen aus dem Data Science Modul
meiner Weiterbildung zur **KI-Entwicklung**.

## Inhalte

| Ordner | Beschreibung |
|---|---|
| `Uebung17301SummeListe` | Performance-Vergleich: Built-in `sum()` vs. manuelle Schleife mit `timeit` |
| `Uebung17302Insertion-Sort` | Insertion Sort – vier Varianten: in-place, funktional, binär, in-place binär |
| `Uebung17303-binaereSuche` | Binäre Suche – rekursiv und iterativ |
| `Uebung22102LineareRegression` | Lineare Regression von Grund auf mit Korrelationskoeffizient |

| `Uebung27302GiniGain` | Gini-Gain für Entscheidungsbaum-Attributauswahl |
| `Uebung28301RandomForest` | Decision Tree und Random Forest Vergleich |
| `Uebung29402KMeans` | K-Means mit 2, 3 und 4 Clustern visualisiert |
| `Uebung29503KMeansUndProzess` | Vollständiger KMeans-Prozess auf Kaggle-Datensatz mit Elbow und Silhouette |

| `CaseStudy202` | Handy-Preisklassifikation – Random Forest, KNN, SVM, Logistische Regression |

## Highlights
- Eigene k-Fold Kreuzvalidierung ohne sklearn
- Eigene Polynomfeature-Generierung
- Binäre Suche rekursiv und iterativ
- Vier Insertion Sort Varianten im Vergleich
- Vollständiger KMeans-Prozess mit Elbow und Silhouetten-Koeffizient
- 3D-Clustering-Visualisierung mit drei Features
